Creating a Contract Evaluation System (CES), utilising a synthetic dataset for NHS necessities (healthcare) with realistic quoted and actual prices as well as quoted and actual delivery times.

- Pipette Tips               
- Reagent Bundles            
- Sharps Disposal            
- Blood Collection Kits      
- Urine/Stool Sample Kits    
- PCR Test Kits

Machine models used to classify if the quoted and actual values for:
- Quoted Time = Actual Time
- Quoted Cost = Actual Cost

Following classification the secondary functions designed to predict maximum values for either to act as most realistc result based on probabilities of quoted and actual values matching. 
System designed to better analyse risk and project real time figures based on a array of features to select the optimum bid based on:
1. Cheapest price
2. Quickest Delivery Time
3. Most Reliable to Quoted Values
