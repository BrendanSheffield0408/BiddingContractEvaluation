Creating a Contract Evaluation System (CES), utilising a synthetic dataset for NHS necessities (healthcare) with realistic quoted and actual prices as well as quoted and actual delivery times.

- Pipette Tips               
- Reagent Bundles            
- Sharps Disposal            
- Blood Collection Kits      
- Urine/Stool Sample Kits    
- PCR Test Kits

Acting as a prelimiary System. 
Machine model developed to predict:
1. likelihood of quoted price == actual price 
2. likelihood quoted timeframe == actual timeframe 

 These predictions will later be integrated into a system to order bids based on a calculation of both the predicted and quoted features.
